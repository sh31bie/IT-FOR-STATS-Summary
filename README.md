TOPIC 1: INTRODUCTION TO IT AND STATS

IT is the use of computers, networks, softwares and systems to gather store, analyze, model and communicate data.
Statistics are raw facts
Software, is a set of instructions that tell a computer what to do and how to do it.
Information is processed data.

Application of IT

Processing data
Analyzing/processing data
Storing data
Presenting data in an organised way.

Role of IT in the society

Transport
Used in GPS software and controlling traffic lights.

Security
Used in surveillance

Education
Used as a way to access learning materials.

Agriculture
Track crop growth, weather patterns and soil health.

Data VS Information
Data is is a raw fact/statistic and is often the input.
Information is processed data and is often the output.

A computer is an electonic device that processes and stores data.

Types of computers.
Supercomputers - they are huge, powerful and are used for complex calculations.
Mainframe - large and are used to process complex data.
Minicomputers - small and used in small businesses.
Microcomputers/PCs - are the personal computers for daily use.

Characteristic.
Speed - does tasks very fast.
Accuracy - provides accurate info most times.
Automation - can work on its own once given instructions.
Storage - can store huge amounts of data.
Versatility - can do many different tasks.
Connectivity - can connect to other networks or computers to share/receive data.
